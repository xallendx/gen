# GenLayer Event Alarm

A simple and modern event alarm system for GenLayer community events. Track weekly events, set alarms, and never miss your favorite events.

## Features

- Weekly event schedule with countdown timers
- Alarm notifications for upcoming events
- Role system information
- Dark/Light theme support
- Mobile responsive design
- Multi-language support (Indonesian & English)

## Tech Stack

- Next.js 16 with App Router
- TypeScript
- Tailwind CSS
- shadcn/ui Components
- Framer Motion

## Getting Started

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build
```

## Deployment

This app is ready to deploy on Vercel, Netlify, or any platform that supports Next.js.
